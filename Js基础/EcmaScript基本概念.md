# ECMAScript基础
 ## 语法
  ### 区分大小写
  ```javascript
     //  变量声明的时候   Js是区分大小写的  比如
       var UserName='赵四'
       var username='刘能'
       console.log(UserName)   // 赵四
       console.log(username)  //刘能
    //变量 UserName和username分别表示两个不同的变量
  ```
